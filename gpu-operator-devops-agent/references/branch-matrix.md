# GPU Operator Product-Specific Branch Matrix

Generated from a curated source manifest. This file captures
the product/operator judgement a fresh agent should not need to rediscover
on each run.

Use this matrix before selecting install, maintenance, troubleshooting, or
recovery branches. A branch is not ready for the default path unless it has a
matching command card and, for fragile shell sequences, a runnable script.
