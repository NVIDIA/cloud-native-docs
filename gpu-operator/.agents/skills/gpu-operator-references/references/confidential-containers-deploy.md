The NVIDIA GPU Operator supports deploying Confidential Containers using Kata Containers and NVIDIA Reference Architecture for Confidential Containers.

Documentation for configuring the GPU Operator for Confidential Containers is available in the [NVIDIA Reference Architecture for Confidential Containers documentation](https://docs.nvidia.com/datacenter/cloud-native/confidential-containers/latest/overview.html).
